A **Data Science project** that segments e-commerce customers using **RFM Analysis (Recency, Frequency, Monetary)** and **K-Means Clustering**.
The goal is to identify different types of customers based on purchasing behavior to help businesses improve marketing and retention strategies.

---

## 🚀 Project Overview

Customer segmentation is an important technique used in **marketing analytics** to understand customer behavior.

In this project:

* Customers are analyzed using **RFM metrics**
* Data is standardized and clustered using **K-Means**
* Different customer segments are identified for targeted strategies

---

## 🧠 RFM Metrics

| Metric        | Meaning                                 |
| ------------- | --------------------------------------- |
| **Recency**   | How recently a customer made a purchase |
| **Frequency** | How often the customer purchases        |
| **Monetary**  | Total money spent by the customer       |

These three metrics help identify **high-value vs low-value customers**.
